代理相关说明

一、代理条件：

    在厂家直供专区任意消费一次，即可成为终身代理。


二、代理优势

    1、不用每天在群里发单，分享海报即可终身锁定粉丝，一劳永逸。

    2、粉丝自主购物比在群里领券更加方便。

    3、可以组建自己的代理商团队，把市场做得更大，获得不在职收入。

    4、项目长久稳定，没有泡沫，跟着淘宝，靠它养老。

    5、投资小，仅仅是一次消费即可获得一个创业机会。

    6、零风险，通过自身购物获取商家佣金，也能收回投资。

    7、回本快，既可招代理获得推广奖励，也可赚商家佣金。

    8、运营成本低，不用给代理商发工资，每月赚的都是纯利润。

    9、不用屯货，不用发货，不用售后服务。

    10、自由职业，时间自由，既能赚钱，也能顾家。

    11、可融入社群，整合人脉，整合资源，边玩边赚。

    12、多方共盈。消费者省钱，推广者赚钱，商家提升销量，淘宝获得用户。

    13、粉丝购买量大，淘宝和天猫每月活跃用户达5亿多，超过其他平台的总和。


三、分配制度

    1、招商奖励

    直接招募一个代理，可获得招商奖励100元；从第7个起，奖励160元。

    间接招募一个代理，可获得招商奖励100元。

    2、商家佣金

    自己领券，你可获得商家佣金40%。

    自己锁定的粉丝领券，你可获得商家佣金40%。

    直接代理锁定的粉丝领券，你可获得商家佣金20%。

    间接代理锁定的粉丝领券，你可获得商家佣金10%。


四、收益分析

        如果招募到一个独家总代理，等同于直接招募了一个一级代理，间接招募了100个二级代理。你可以获得10100元推广奖励（详见独家总代理政策）。

        如果平均每三天招募一个398代理，则一年可以招募至少100个一级代理。你可以获得推广奖励16000元。

        如果你的100个一级代理各招募20个代理，你就有2000个二级代理，你可以获得推广奖励20万元。

        如果你的2000个代理每人锁定100个粉丝，总计粉丝可达20万人。据趣味荟现有数据统计，平均每个粉丝领券，所产生的商家佣金约为11元/月。20万粉丝的总佣金达220万。你的佣金提成10%~20%，你的佣金提成为22万~44万/月。
